# Hardware заметки

Сюда добавляем схемы, распиновки, BOM и фото сборок для разных каналов связи и плат (Nano, Pi Zero, ESP32-CAM/Dev, Pico). Каждую новую конфигурацию описываем отдельным файлом или подпапкой.

## Минимальные требования по питанию и интерфейсам
- **Arduino Nano** — питание 5 В, логика 5 В; связь с контроллером по UART/I2C для простых датчиков или реле.
- **Raspberry Pi Zero** — питание 5 В 2 A, общая земля с FC; может работать как шлюз (PPP/NCM для LTE, маршрутизатор Wi‑Fi).
- **ESP32-CAM / ESP32-Dev** — питание 5 В с запасом по току; UART↔UDP мост для телеметрии или видеопоток MJPEG/RTSP.
- **Raspberry Pi Pico** — питание 5 В/3.3 В, логика 3.3 В; SPI/I2C/UART для вспомогательных датчиков или PWM.

## Что фиксировать в новых файлах
- Схему подключения (питание, уровни логики, пины UART/I2C/SPI) и фото сборки.
- Используемый модем/радио, версия прошивки/скетча и скорость портов.
- Особенности охлаждения и размещения антенн, чтобы повторять испытания.
