Справочник команд

            

              

                
ARM / DISARM

                

                  
ARM: включает моторы/активацию (если автопилот разрешит).

                  
DISARM: выключает.

                

              

              

                
SET_MODE

                

                  
MANUAL: ручное управление (обычно принимает RC override).

                  
HOLD: удержание.

                  
AUTO: миссия.

                

              

              

                
RC_OVERRIDE

                

                  
Отправляет RC_CHANNELS_OVERRIDE.

                  
steering_pwm: 1000..2000 (CH1), 1500 центр.

                  
throttle_pwm: 1000..2000 (CH3), 1500 стоп.

                  
Нужно удерживать (мы шлём каждые ~200ms, пока держишь кнопку).

                

              

              

                
COMMAND_LONG

                

                  
Универсальная команда: MAV_CMD + 7 параметров.

                  
Пример ARM: MAV_CMD_COMPONENT_ARM_DISARM = 400, param1=1.

                  
Пример REBOOT: MAV_CMD_PREFLIGHT_REBOOT_SHUTDOWN = 246, param1=1.
